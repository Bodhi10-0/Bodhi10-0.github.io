C- Swap

Rules
  1. The programme consist of two sections.
  2. The first section is the preliminary test and the second section is the coding section.
  3. The team has to pass the preliminary test to qualify for the second section.
  4. Each team should contain only two members.

Rules for coding section:
  1. Time alloted for each question is 30 minutes.
  2. The first member of the team has to start coding the program while the second member will start coding the second problem.

  4. Communication between the team members are not allowed.
  5. Comments are not allowed in the program code.
  6. Decisions made by the programme coordinators will be the final.
